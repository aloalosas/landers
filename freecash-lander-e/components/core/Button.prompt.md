Freecash's signature pulsing-green CTA — use it as the single primary action on a lander step.

```jsx
<Button href="https://your-referral-link" arrow>Download &amp; Start Earning</Button>
```

Variants:
- `variant="primary"` — neon-green pulse + sheen sweep (default). White uppercase label.
- `variant="gold"` — viral/scarcity theme, Montserrat, gold gradient with hard bottom shadow.
- `variant="apple"` — Apple-Pay theme, solid blue, no pulse, sentence case.
- `size` — `lg` (default, 22px pad) · `md` · `sm`.
- `arrow` — adds the trailing → glyph. `href` makes it an `<a>` (preserve real referral/tracking URLs); omit for a `<button>`.
